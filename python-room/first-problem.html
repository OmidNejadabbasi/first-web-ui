2. Let's get this done!

The problem gives you a number and you have to print its inverse. Like if it gives you 1234, you have to print 4321. But notice that you can do this easily in Python with only one line of code :

print(input()[::-1])

But we want you to convert the input to a integer and then do some math in it then reverse it.

3. The answer!

First I am going to show you the code then I will explain how it works!

n = int(input('Enter the number: '))
reverse = 0
while n > 0:
    first_digit = n % 10
    reverse = (reverse * 10) + first_digit
    n = n // 10
                
print(reverse)

First, we define a variable named reversed to hold the result. We update it on every step of the loop. Then we enter a while loop with the condition that n should be greater than 0.

At first line of the loop we store the first digit of n in the variable first_digit, by obtaining its modulo 10. Then we add it as the next digit to reversed . Now that we are done with the first digit of n, we have to take it out. We do it by floor division operator. When the number n runs out of digits it is no longer greater than 0 and we exit the while loop. Then print reverse!
